<li>推牆演算</li>
為防止每次重新刷新牆壁的時候角色會卡在牆裡面
需要對於玩家現在所處的位置是否真的為可存在之處(非牆壁、非王、沒有超出地圖)
做檢查，若有違反就要推回到目前位置最近的安全地點。
<li>跳躍演算</li>
空中可以控制左右方向，但是不能二次跳躍。
使用bool紀錄玩家現在的跳躍狀態，直到完成一次跳躍之後，才能再次跳躍。
對於跳躍完成的判定，"腳下現在是可以踩的地方"。
如果地板空了，就要繼續往下掉，所以隨時都要檢查腳下的物件是否為地板。
<li>王的位移追蹤</li>
